# Spam Email Classifier

This project is a **Spam Email Classifier** that detects whether an email message is **Spam** or **Not Spam (Ham)** using **Natural Language Processing (NLP)** and **Naive Bayes Classifier**.



- Python 3.x
- NLTK (Text preprocessing, stopwords, stemming)
- Scikit-learn (TF-IDF Vectorizer, Naive Bayes)
- Streamlit (Interactive Web App)
- Pandas & NumPy (Data handling)
